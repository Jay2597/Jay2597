### 🛠 Featured

- **[Vanara](https://vanaraagents.com)** — a depth-gated catalog of 200+ specialist AI agents, skills & outcome packs for Claude Code, installable with `npx vanara install <item>`. Runs on your own Claude subscription — no API keys, no metering. Open-core.
  · Free tier → **[vanara-agents/skills](https://github.com/vanara-agents/skills)**
- **[clawprint](https://github.com/vanara-agents/clawprint)** — the capability manifest for your `.claude/` directory. Know exactly what your agent setup can do, and catch every change in PR review. Deterministic, zero-dependency, no AI, no verdicts.
- **[Groundwork](https://jay2597.github.io/groundwork/)** — a local-first, client-side design tool. No cloud; your files stay on your device.
  · [repo](https://github.com/Jay2597/groundwork)
- **WeEven** — a React Native + TypeScript + Supabase mobile app. → [we-even.com](https://we-even.com)
- **Fintech & trading** — [FinSight](https://github.com/Jay2597/finsight-demo) (portfolio-intelligence terminal), [crypto-live-scan](https://github.com/Jay2597/crypto-live-scan) (15-min crypto trigger scanner on GitHub Actions), [nifty-condor-paper](https://github.com/Jay2597/nifty-condor-paper) (VIX-gated NIFTY iron-condor paper trading).
